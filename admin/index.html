How to update this site Simple steps for text, artwork, colours, and fonts
  1. Change the visible wording below for headings, sections, questions, or contact details.
  2. Choose a site theme and font style to switch the visual direction for both the public site and this editor.
  3. To change an image, click into an artwork field first, then upload a file or press Use image in the library.
  4. Press Save website changes and rebuild, then open View site to check the live preview.
