Find a parish

Use trusted diocesan routes first, so you are sent to the parish information that local Catholic authorities actually maintain.

Home / Find a Parish

This page no longer keeps a local parish table. Parish information changes too often for a small independent directory to copy faithfully, so the safer route is to point you toward official diocesan websites and trusted Catholic search routes instead.

If you already know the diocese, start there. If you only know a town, island, or parish name, use the trusted search below and open the official diocesan domain that serves the place you need.

Best way to use this page

  • Search by town, city, island, or parish name.
  • Choose a diocese when you already know the local Catholic jurisdiction.
  • Use the diocesan site for current Mass times, parish offices, and sacramental information.
