Catholic Business Directory

A place to find Catholic publishers, shops, makers, and pilgrimage travel across the UK and Ireland.

This directory exists to help Catholics find Catholic businesses, makers, publishers, shops, and services across the UK and Ireland. In an age of anonymous global buying, it is a small way of supporting one another more locally, more ethically, and with a clearer sense of stewardship.

It is also meant to make practical Catholic life easier: finding a missal, ordering a rosary, buying books from a Catholic publisher, booking pilgrim travel, or simply choosing to support a business that is trying to serve the faith honestly.

Reference listing only: a listing here does not imply endorsement by a parish, diocese, bishops' conference, or the Holy See. Where any stronger reference exists, it should be stated plainly in the listing itself.

About this directory

This directory includes businesses owned or run by Catholics, businesses serving Catholic life, businesses offering goods or services relevant to Catholic practice, and a small number of ecclesially referenced entries where that connection is plain from the evidence.

Not all listings mean the same thing. Each record therefore carries a category and a verification level so readers can tell the difference between an ordinary standard listing, an editorially reviewed entry, and any stronger reference that may later be documented.

The working standards for inclusion, exclusions, and correction handling are set out more fully on the Directory policy page.

How to read the directory

  • Catholic-owned: owned or run by Catholics where that connection is part of the listing basis.
  • Catholic-serving: businesses serving Catholic life without implying Catholic ownership.
  • Catholic goods/services: businesses offering goods or services directly relevant to Catholic practice.
  • Ecclesially referenced: used only where a stronger ecclesial connection is clearly evidenced.

Submit a business

If you run a business that belongs here, or know one that does, use the submission page so the directory can be kept careful, clear, and fair. A donation is warmly appreciated because it helps keep the directory reviewed and useful, but submission and support are not the same thing.

Submission and review

Every listing should be capable of being explained in plain terms: why it belongs in a Catholic directory, which category fits it best, what evidence supports it, and whether any higher reference level can actually be shown. If that cannot be stated clearly, the listing should not be strengthened by vague language.
